transformBmp
=====

##### by BEJL (Bill, Jordan, Ender, Liza)
-----

## Documentation:

### Operation:

To use the `transformBmp` node command:

```
$ node transformBmp <bmp origin filepath> <transformation> <bmp destination filepath>
```
Valid transformations include:
```
neon
invert
black
darkenDarks
```

For help, type:
```
$ node transformBmp help
```
